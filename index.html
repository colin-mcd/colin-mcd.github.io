Colin McDonald

Third-year PhD student at Carnegie Mellon University advised by Umut Acar.

My broad research interests include compilers and programming languages, parallelism, type theory, NLP, and formal logic. I'm currently working on compiling efficient parallel functional programs by integrating custom control-flow primitives into the MPL compiler.

You can check out what I've been up to recently on my GitHub page.
